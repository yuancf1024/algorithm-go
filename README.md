**2022-04-19C++?**

今天参加了百度的实习生笔试，然后算法题我用的是Go语言，其中第2、3题都遇到了**超时**的问题，第2题通过添加`break`语句，成功解决。而第3题，通过测试90%，但是依然超时，绞尽脑汁，在本地和OJ反复调试了很多遍，还是没能够找出超时的原因😂。情急之下，无奈的将代码迅速用C++改写，提交，通过测试85%。

晚上在QQ群和大佬们交流后才发现：牛客OJ对Go不友好，很容易产生超时问题，建议C++。

于是乎，下定决心，C++得捡起来啦!加油💪
